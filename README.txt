About the project:
This project is cater to people who may want to detect themselves for phobia 
and a self-trainer at their comfort. Hopefully, more people are willing to train themselves 
and improve their mental health over time. Currently only Scopophobia is added.

STEPS TO RUN THE PROJECT:
1) Run Python file Counter_Phobia.py
2) This project is on Flask micro web framework, so please look at the VS Code IDE terminal 
and look for "Running on http://127.0.0.1:5000", Ctrl + Click with the cursor to open the website
3) Please be aware that this project uses openCV, so camera will be turn on
4) If CSS styling cannot be seen, refresh the Website by using Ctrl + Shift + R
5) Laptop with embedded webcam is a must
